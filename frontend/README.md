# BookMyStay Frontend

Frontend for BookMyStay hotel booking platform.

## Pages

- `index.html` - Homepage
- `listings.html` - Hotel listings
- `hotel.html` - Hotel details
- `booking.html` - Booking page
- `feedback.html` - Feedback form
- `login.html` - Login/Signup

## Setup

1. Update API URL in all HTML files
2. Open `index.html` in browser
